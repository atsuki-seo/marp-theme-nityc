# 現行システムの課題

## 主な問題点
### 技術的課題
- 古いシステム基盤（10年前の技術）
- 保守性の低下

> 現行システムは2014年に構築されており、技術的負債が蓄積しています

```sql
-- 例：古いデータベース構造
SELECT * FROM student_info 
WHERE status = 'active';
```

これらの課題を解決するため、新システムへの移行が必要です。

<div class="summary-negative">

- 移行期間中の業務への影響

- 既存データの移行リスク
